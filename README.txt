Crea planos desde Mapplanner en minutos e integralos en tu web en segundos con este plugin para WordPress para planos interactivos.

== Description ==
Mapplanner.co integra tus planos interactivos en tu web  de manera muy sencilla, pero para facilitar aún más la tarea a aquellos que han desarrollado su web con WordPress, hemos creado este plugin.

= Cómo funciona nuestro plugin =

Simplemente descárgalo, actívalo y utiliza este shortcode donde quieras colocar tu plano:

[mapplanner-map account="*" map="*"/]

Los dos parámetros pasados al shortcode indican el número identificador de tu cuenta y el número identificador del plano que quieres incrustar.

Tranquilo, no es necesario que conozcas estos valores, si entras en tu cuenta en app.mapplanner.co y accedes a la configuración del plano, encontrarás el código shortcode que debes utilizar para ese plano en concreto.

Una vez tengas tu shortcode colocado en tu página o post, guárdalo y visita tu web, el plano ya debería aparecer en ella.

Si tienes algún problema para insertar los planos, como siempre, puedes contactar en el correo hola@mapplanner.co

== Installation ==
**Installation through the WordPress Dashboard**
1. Login to your WordPress admin.
2. Click on the plugins tab.
3. Click the Add New button.
4. Search for "Mapplanner".
5. Look for our plugin, click "Install Now" and then "Activate".
6. Once the plugin is installed and active, you simply need to get your shortcode from our service. Try for free in http://mapplanner.co
